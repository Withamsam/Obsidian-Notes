# [[dirb]]  
___

## Description:
A web content scanner that runs a dictionary attack on sites to determine if a link is valid.

## Installation
### Kali
`sudo apt update && sudo apt install dirb`

## Commands
**Basic**
`dirb http://10.10.10.10/ /path/to/wordlist -o ./where/file/should/save`

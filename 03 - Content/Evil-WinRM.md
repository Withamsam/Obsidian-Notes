# [[Evil-WinRM]]  
___

## Description:
Tool designed to exploit a users account that has access to **WinRM** group "Remote Management Users"

## Installation
Pre-installed on Kali

## Commands
```bash
evil-winrm -i <VICTIM_IP> -u <USERNAME> -p <USER_PASS>
```

# [[Remmina]]
___

## Description:
Open source remote access screen and file sharing to your desktop

## Installation
### Ubuntu
```bash
sudo apt-add-repository ppa:remmina-ppa-team/remmina-next
sudo apt update
sudo apt install remmina remmina-plugin-rdp remmina-plugin-secret
```

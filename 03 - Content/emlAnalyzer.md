## Installation

## Commands
`emlAnalyzer -i Urgent\:.eml --header --html -u --text --extract-all`

| Query Details |       Explanation       |
|:-------------:|:-----------------------:|
|      **-i**       | Path to file to analyze |
|   **--header**    |       Show header       |
|      **-u**       |       Show URL's        |
|    **--text**     |     Show cleartext      |
| **--extract-all** | Extract all attachments |
